Formulas

SoC window = max(0, initial SoC − final SoC) usable Ah = capacity Ah × SoC window avg voltage = ∫ V(SoC) dSoC over SoC window (trapezoidal) usable Wh = usable Ah × avg voltage constant current: runtime h = usable Ah ÷ current A constant power: runtime h = usable Wh ÷ power W split duty: each phase contributes (duty % × phase current A) ÷ 100; CP phases are converted to current via avg voltage
