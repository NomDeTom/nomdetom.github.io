Discord Timestamp Generator

Generate formatted timestamps for your Discord messages

Back to index

Use Discord's timestamp feature to display dates and times that automatically adjust to each user's local timezone. Select a date and time, choose your preferred format, and copy the code to paste in Discord messages.

Format Guide

Select the components and style you want:

  • Date: Include a date in the timestamp
  • Time: Include a time in the timestamp
  • Day of Week: Add the day name (only shown with Date + Time)
  • Short: Use compact formatting (12/20/2025, 2:30 PM)
  • Long: Use full formatting (December 20, 2025, 2:30:45 PM)
  • Relative: Show relative time like "2 hours ago" (incompatible with other options)

Note: When both Date and Time are selected, Discord only offers the Long Date format combined with Short Time format (the Short/Long toggle is not available in this case).
