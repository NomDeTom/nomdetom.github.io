Inputs

Run length
0.017241 Ohm mm²/m

Outputs

Conductor resistance (per meter, single core) 0 Ohm/m
Total loop resistance 0 Ohm
Voltage drop 0 V
Voltage drop percent 0 %
Load voltage 0 V
Cable power loss 0 W

Values are estimates at DC / low frequency.

Formulas

Base resistivity input: ρ20 in Ω·mm²/m at 20 °C Temperature-adjusted resistivity: ρ(T) = ρ20 × (1 + α × (T - 20 °C)) Temperature coefficient: α in 1/°C Conductor resistance per meter: Rm = ρ(T) / A Total resistance: Rtotal = Rm × length factor × run length Voltage drop: Vdrop = I × Rtotal Load voltage: Vload = Vsupply - Vdrop Cable loss: Ploss = I2 × Rtotal
