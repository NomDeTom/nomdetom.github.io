ASCII IC Pinout Tool Reference

Usage notes for ic_pinout_ascii.py, plus verified example commands and outputs for ATtiny412 and ATtiny816 packages.

Back to index

Reference package pinouts were checked against Microchip product documentation for ATtiny412 and ATtiny816.

Source pages: ATtiny412 8-pin package and ATtiny816 20-pin VQFN.

Overview

The tool generates ASCII package drawings for:

  • 2-sided packages with two
  • 4-sided packages with four
  • Compact 4-sided layouts with --vertical-spread
For 2-sided packages, pass left pins top-to-bottom and right pins top-to-bottom. For 4-sided packages, pass top left-to-right, right top-to-bottom, bottom left-to-right, and left top-to-bottom.

Basic Usage

python ic_pinout_ascii.py two --name "IC_NAME" --left "PIN1,PIN2" --right "PINN,PINN-1"
python ic_pinout_ascii.py four --name "IC_NAME" --top "A,B,C" --right "D,E,F" --bottom "G,H,I" --left "J,K,L"
python ic_pinout_ascii.py four --vertical-spread --name "IC_NAME" --top "A,B,C" --right "D,E,F" --bottom "G,H,I" --left "J,K,L"

Reference Commands

Pinouts verified against Microchip online documentation: ATtiny412 8-pin package and ATtiny816 20-pin VQFN. 

python ic_pinout_ascii.py two --name "ATtiny412" --left "VDD,PA6,PA7,PA1" --right "GND,PA3/EXTCLK,PA0/RESET/UPDI,PA2"
python ic_pinout_ascii.py four --name "ATtiny816" --top "PA1,PA0/RESET/UPDI,PC3,PC2,PC1" --right "PC0,PB0,PB1,PB2/TOSC2,PB3/TOSC1" --bottom "PA5,PA6,PA7,PB5,PB4" --left "PA2,PA3/EXTCLK,GND,VDD,PA4"
python ic_pinout_ascii.py four --vertical-spread --name "ATtiny816" --top "PA1,PA0/RESET/UPDI,PC3,PC2,PC1" --right "PC0,PB0,PB1,PB2/TOSC2,PB3/TOSC1" --bottom "PA5,PA6,PA7,PB5,PB4" --left "PA2,PA3/EXTCLK,GND,VDD,PA4"
