Formulas

Symbol time: T_sym (ms) = 2^SF / BW_kHz Preamble duration: T_pre = (preamble_len + 4.25 sync word) × T_sym — the +4.25 is the LoRa sync word overhead (always present, independent of header mode) Payload bits: 8×PL − 4×SF + 8 + 16×CRC + 20×(explicit header) ≥ 0 — the LoRa header overhead (20 bits) is encoded in the first payload codewords, not the preamble Bits per symbol: SF − 2 (LDRO on) or SF (LDRO off) Payload symbols: ceil(payload_bits / 4 / bits_per_sym) × CR_denom + 8 Overhead symbols: 4.25 sync word — appended to the configured preamble length; appears in every packet, independent of payload size or header mode Time on Air: T_pre + payload_symbols × T_sym LDRO auto: enabled when T_sym > 16 ms (SX127x/SX126x requirement) Meshtastic limits (from airtime.h): hard limit 40% channel utilisation/hour; polite limit 25% Formula source: Semtech SX127x/SX126x datasheet; cross-referenced with ifTNT/lora-air-time Rx sensitivity: datasheet figures used for SX1276/8, SX1262, LLCC68, LR1121, LR20xx (RxBoosted, CR=4/5, PER=1%). Intermediate SFs interpolated linearly between datasheet anchors; BW correction 10 log₁₀(BW_selected / BW_anchor) applied when BW is not in the table. SX1280 and LR1110 use the thermal noise formula (−174 + 10 log₁₀(BW) + NF + SNR_min) with a chip-appropriate NF. “All / Generic” uses the formula with the user-supplied NF value. EIRP: Tx power + Tx antenna gain Link budget: EIRP + Rx antenna gain − Rx sensitivity — total dB available to overcome path loss Max range: free-space path loss model — FSPL (dB) = 20 log₁₀(d_km) + 20 log₁₀(f_MHz) + 32.45; solve for d when FSPL = link budget
