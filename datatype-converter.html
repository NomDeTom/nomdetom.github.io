Notes

Signed integer rows use strict range validation (out-of-range values are not wrapped in this version). float32/float64 rows include sign/exponent/mantissa bit slices in the Notes column. Changing endian reverses byte order for multi-byte datatypes; bit fields are still shown in canonical big-endian bit order.
